MicroService

1. Tech stack.
  ![Tech stack](micro.png)


   - Spring Cloud : spring eureka discovery/registry, feign, hytrix, dashboard, spring config, spring auth services and security.
   - Apache kafka
   - ELK (elasticsearch, logstask, kibana) for centralized logging
   - Docker
